**Interactive Neural Network Decision Boundary Visualizer** — A browser-based educational platform that visualizes how neural networks learn to classify 2D data points in real-time. Built with **Hexagonal Architecture** (Ports & Adapters) to demonstrate clean separation of concerns and enterprise-grade software design patterns.
